# Instruções rápidas para publicar no GitHub

1. Acesse github.com e faça login.
2. Clique em **New repository**.
3. Nome do repositório: `flui-app-prototype`.
4. Marque como **Public**.
5. Clique em **Create repository**.
6. Envie todos os arquivos desta pasta para o repositório.
7. Depois de subir, copie o link do repositório e cole no PDF ou no campo de entrega da faculdade.

## Para ativar GitHub Pages

1. Entre no repositório.
2. Vá em **Settings**.
3. Clique em **Pages**.
4. Em Source, selecione a branch `main`.
5. Selecione a pasta `/root`.
6. Salve.
7. O GitHub vai gerar um link público para abrir o protótipo.

Depois disso, o link do protótipo poderá ser colocado no PDF de entrega.
